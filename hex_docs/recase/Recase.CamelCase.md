# Recase.CamelCase

Module to convert strings to `camelCase`.

This module should not be used directly.

## Examples

    iex> Recase.to_camel "foo_barBaz-őĽambdaőõambda-–Ņ—Ä–ł–≤–Ķ—ā-–ú–ł—Ä"
    "fooBarBazőõambdaőõambda–ü—Ä–ł–≤–Ķ—ā–ú–ł—Ä"
