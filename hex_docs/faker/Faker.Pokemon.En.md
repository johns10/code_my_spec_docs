# Faker.Pokemon.En

Functions for Pokemon names in English

## location()

Returns a location from Pokemon universe

## Examples

    iex> Faker.Pokemon.En.location()
    "Vaniville Town"
    iex> Faker.Pokemon.En.location()
    "Slateport City"
    iex> Faker.Pokemon.En.location()
    "Shalour City"
    iex> Faker.Pokemon.En.location()
    "Solaceon Town"

## name()

Returns a Pokemon name

## Examples

    iex> Faker.Pokemon.En.name()
    "Fraxure"
    iex> Faker.Pokemon.En.name()
    "Shellos"
    iex> Faker.Pokemon.En.name()
    "Ambipom"
    iex> Faker.Pokemon.En.name()
    "Forretress"
