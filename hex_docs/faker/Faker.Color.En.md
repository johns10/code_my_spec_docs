# Faker.Color.En

Functions for color data in English

## fancy_name()

Returns a random English fancy color name

## Examples

    iex> Faker.Color.En.fancy_name()
    "Tawny"
    iex> Faker.Color.En.fancy_name()
    "Citrine"
    iex> Faker.Color.En.fancy_name()
    "Greige"
    iex> Faker.Color.En.fancy_name()
    "Cesious"

## name()

Returns a random English color name

## Examples

    iex> Faker.Color.En.name()
    "Red"
    iex> Faker.Color.En.name()
    "Green"
    iex> Faker.Color.En.name()
    "Brown"
    iex> Faker.Color.En.name()
    "Pink"
