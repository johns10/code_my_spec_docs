# Faker.Internet.It

Generating internet related data in Italian

## domain_suffix()

Returns a random domain suffix

## Examples

    iex> Faker.Internet.It.domain_suffix()
    "com"
    iex> Faker.Internet.It.domain_suffix()
    "it"
    iex> Faker.Internet.It.domain_suffix()
    "com"
    iex> Faker.Internet.It.domain_suffix()
    "biz"

## free_email_service()

Returns a random free email service name

## Examples

    iex> Faker.Internet.It.free_email_service()
    "virgilio.it"
    iex> Faker.Internet.It.free_email_service()
    "yahoo.it"
    iex> Faker.Internet.It.free_email_service()
    "aruba.it"
    iex> Faker.Internet.It.free_email_service()
    "gmail.com"
