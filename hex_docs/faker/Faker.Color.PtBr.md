# Faker.Color.PtBr

Functions for color data in Brazilian Portuguese

## name()

Returns a random Brazilian Portuguese color name

## Examples

    iex> Faker.Color.PtBr.name()
    "Vermelho"
    iex> Faker.Color.PtBr.name()
    "Verde"
    iex> Faker.Color.PtBr.name()
    "Marrom"
    iex> Faker.Color.PtBr.name()
    "Rosa"
