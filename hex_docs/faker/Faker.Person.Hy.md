# Faker.Person.Hy

Functions for generating name related data in Armenian

## first_name()

Returns a random first name.

## Examples

    iex> Faker.Person.Hy.first_name()
    "Սյուզաննա"
    iex> Faker.Person.Hy.first_name()
    "Վարուժան"
    iex> Faker.Person.Hy.first_name()
    "Բարբարա"
    iex> Faker.Person.Hy.first_name()
    "Սոֆյա"
