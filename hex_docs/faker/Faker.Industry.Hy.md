# Faker.Industry.Hy

Functions for generating industry related data in Armenian

## industry()

Returns an industry name.

## Examples

    iex> Faker.Industry.Hy.industry
    "Հյուրընկալություն"
    iex> Faker.Industry.Hy.industry
    "Բժշկական Գործունեություն"
    iex> Faker.Industry.Hy.industry
    "Վենչուրային և Մասնավոր Կապիտալ"
    iex> Faker.Industry.Hy.industry
    "Էներգետիկա"
