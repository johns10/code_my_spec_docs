# Faker.Color.Hy

Functions for generating color data in Armenian

## name()

Returns a random color name.

## Examples

    iex> Faker.Color.Hy.name()
    "մոխրագույն"
    iex> Faker.Color.Hy.name()
    "կանաչ"
    iex> Faker.Color.Hy.name()
    "երկնագույն"
    iex> Faker.Color.Hy.name()
    "մանուշակագույն"
