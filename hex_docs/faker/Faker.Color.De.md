# Faker.Color.De

Functions for color data in German

## fancy_name()

Returns a random english fancy color name

## Examples

    iex> Faker.Color.De.fancy_name()
    "Flieder"
    iex> Faker.Color.De.fancy_name()
    "Feldgrau"
    iex> Faker.Color.De.fancy_name()
    "Gelbgrün"
    iex> Faker.Color.De.fancy_name()
    "Rotbraun"

## name()

Returns a random German color name

## Examples

    iex> Faker.Color.De.name()
    "Rot"
    iex> Faker.Color.De.name()
    "Grün"
    iex> Faker.Color.De.name()
    "Braun"
    iex> Faker.Color.De.name()
    "Rosa"
