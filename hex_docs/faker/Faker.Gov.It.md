# Faker.Gov.It

Functions for data created/released by the Italian government,
like fiscal codes

## fiscal_id()

Returns a random Italian fiscal code

## Examples

    iex> Faker.Gov.It.fiscal_id
    "ELRCEA64C50A918F"
    iex> Faker.Gov.It.fiscal_id
    "ZSLNKH22M34H480J"
    iex> Faker.Gov.It.fiscal_id
    "OCPCVO90M50F353I"
    iex> Faker.Gov.It.fiscal_id
    "PQYRFX94R54C681K"
