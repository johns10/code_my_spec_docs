# Faker.Aws.PtPt

Functions for generating AWS information in Portuguese of Portugal

## region_name()

Returns a random region name available on AWS

## Examples

    iex> Faker.Aws.PtPt.region_name()
    "Asia Pacifico (Tóquio)"
    iex> Faker.Aws.PtPt.region_name()
    "EUA Este (Ohio)"
    iex> Faker.Aws.PtPt.region_name()
    "Europa (Milão)"
    iex> Faker.Aws.PtPt.region_name()
    "Africa (Cape Town)"
