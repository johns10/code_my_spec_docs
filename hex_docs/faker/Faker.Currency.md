# Faker.Currency

Functions for generating currency related data

## code()

Returns a random currency code

## Examples

    iex> Faker.Currency.code()
    "WST"
    iex> Faker.Currency.code()
    "UYU"
    iex> Faker.Currency.code()
    "CRC"
    iex> Faker.Currency.code()
    "DOP"

## symbol()

Returns a random currency symbol

## Examples

    iex> Faker.Currency.symbol()
    "£"
    iex> Faker.Currency.symbol()
    "฿"
    iex> Faker.Currency.symbol()
    "ƒ"
    iex> Faker.Currency.symbol()
    "Rp"
