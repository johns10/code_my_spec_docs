# Faker.Superhero

Functions for generating Superhero data

## descriptor()

Returns a random Superhero descriptor

## Examples

    iex> Faker.Superhero.descriptor()
    "Ronin"
    iex> Faker.Superhero.descriptor()
    "Azrael"
    iex> Faker.Superhero.descriptor()
    "Beyonder"
    iex> Faker.Superhero.descriptor()
    "Phantom"

## name()

Returns a random Superhero name

## Examples

    iex> Faker.Superhero.name()
    "Red Beyonder the Hunter"
    iex> Faker.Superhero.name()
    "Penance Strike"
    iex> Faker.Superhero.name()
    "Sage"
    iex> Faker.Superhero.name()
    "Giant Aqua I"
