# Faker.Internet.En

Generating internet related data in English

## domain_suffix()

Returns a random domain suffix

## Examples

    iex> Faker.Internet.En.domain_suffix()
    "com"
    iex> Faker.Internet.En.domain_suffix()
    "org"
    iex> Faker.Internet.En.domain_suffix()
    "name"
    iex> Faker.Internet.En.domain_suffix()
    "info"

## free_email_service()

Returns a random free email service name

## Examples

    iex> Faker.Internet.En.free_email_service()
    "gmail.com"
    iex> Faker.Internet.En.free_email_service()
    "hotmail.com"
    iex> Faker.Internet.En.free_email_service()
    "gmail.com"
    iex> Faker.Internet.En.free_email_service()
    "hotmail.com"
