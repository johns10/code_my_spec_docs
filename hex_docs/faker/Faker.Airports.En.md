# Faker.Airports.En

Functions for generating airports related data in English

## name()

Returns a random airport name

## Examples

    iex> Faker.Airports.En.name()
    "Union Island International Airport"
    iex> Faker.Airports.En.name()
    "St. John's International Airport"
    iex> Faker.Airports.En.name()
    "Jizan Regional Airport"
    iex> Faker.Airports.En.name()
    "Bisho Airport"
