# Faker.Superhero.En

Functions for Superhero data in English

## descriptor()

Returns a random descriptor

## Examples

    iex> Faker.Superhero.En.descriptor()
    "Ronin"
    iex> Faker.Superhero.En.descriptor()
    "Azrael"
    iex> Faker.Superhero.En.descriptor()
    "Beyonder"
    iex> Faker.Superhero.En.descriptor()
    "Phantom"

## name()
