# Faker.Pokemon.It

Functions for Pokemon names in Italian

## location()

Returns a location from Pokemon universe

## Examples

    iex> Faker.Pokemon.It.location()
    "Arenipoli"
    iex> Faker.Pokemon.It.location()
    "Spiraria"
    iex> Faker.Pokemon.It.location()
    "Novartopoli"
    iex> Faker.Pokemon.It.location()
    "Castel Vanità"

## name()

Returns a Pokemon name

## Examples

    iex> Faker.Pokemon.It.name()
    "Magmortar"
    iex> Faker.Pokemon.It.name()
    "Gastly"
    iex> Faker.Pokemon.It.name()
    "Mienshao"
    iex> Faker.Pokemon.It.name()
    "Articuno"
