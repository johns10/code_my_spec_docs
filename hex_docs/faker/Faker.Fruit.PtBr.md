# Faker.Fruit.PtBr

Functions for fruit data in Brazilian Portuguese.

## fruit()

Returns a fruit.

## Examples

    iex> Faker.Fruit.PtBr.fruit
    "Fruta-do-conde"
    iex> Faker.Fruit.PtBr.fruit
    "Bergamota"
    iex> Faker.Fruit.PtBr.fruit
    "Quixaba"
    iex> Faker.Fruit.PtBr.fruit
    "Amora"
