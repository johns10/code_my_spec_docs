# Faker.Cat.PtBr

Functions for Cat names and breeds in Brazilian Portuguese

## breed()

Returns a Cat breed string

## Examples

    iex> Faker.Cat.PtBr.breed()
    "Angorá Turco"
    iex> Faker.Cat.PtBr.breed()
    "Azul Russo"
    iex> Faker.Cat.PtBr.breed()
    "Pelo Curto Brasileiro"
    iex> Faker.Cat.PtBr.breed()
    "Pelo Curto Americano"

## female_name()
