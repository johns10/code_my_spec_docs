# Faker.Person.En

Functions for name data in English

## first_name()

Returns a random first name

## Examples

    iex> Faker.Person.En.first_name()
    "Joany"
    iex> Faker.Person.En.first_name()
    "Elizabeth"
    iex> Faker.Person.En.first_name()
    "Abe"
    iex> Faker.Person.En.first_name()
    "Ozella"
