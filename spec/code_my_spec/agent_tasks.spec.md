# CodeMySpec.AgentTasks

Specialized task modules — codegen, QA, research, integration planning, three amigos, persona research, spex boundary readiness, architecture design, ADRs, component spec / test, BDD specs — that build agent prompts and evaluate output. Each task is a child module surfaced via the local MCP tools server.
